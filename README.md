# NotXianLiao
### 叔叔我们不闲聊~
这是一个Adblock过滤清单，用来屏蔽闲聊(https://www.xianliao.me/)
<br>
为什么我们要做这个?
<br>
因为闲聊使用亚马逊S3储存空间并且使用亚马逊的CDN，亚马逊CDN在中国访问速度很慢，严重拖慢网络速度，而且闲聊一点卵用也没有...
